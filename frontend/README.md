# BVBChat Frontend

Stock Market Chat Assistant built with React, TypeScript, and Vite.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm run dev
```

The app will be available at http://localhost:5173
